A Python-based computer vision tool that turns your webcam into a touchless mouse.

## Features
- **Air Mouse:** Control the cursor with your index finger.
- **Pinch-to-Click:** Pinch thumb and index to click.
- **Smart Scrolling:** Move hand to edges to scroll (with dynamic speed).
- **Virtual Mousepad:** Uses a reduced frame for comfortable movement.

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Run the script: `python Touchless_Controller.py`
